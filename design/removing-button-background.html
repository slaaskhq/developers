Removing the white background from the button

Whe uploading a transparent image to the Slaask button, you may prefer to remove the white background.

Here's a piece of CSS code you can add inside a <style> tag in order to do so:


#slaask-button {
  -webkit-box-shadow: none!important;
  box-shadow: none!important;
}

#slaask-button .slaask-button-image {
  border: 0;
  background-color: transparent;
}
