Enabling Slaask to work with Turbolinks

When using Slaask with Turbolinks, the widget disappears each time the user moves from one page to another.

You can bypass this issue by adding the following code in a <script> tag:


document.addEventListener("page:load", function() {
  _slaask = new slaaskApp();
  _slaask.init('xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx');
});

Then replace the xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx with your Widget Key.
