Opening Slaask from an item/button

To code a button or a menu navigation item on your site that will open the Slaask widget, you have 2 choices:

A. The URL

To automatically open the Slaask widget when someone lands on a page, please add "#chat-open" to the end of your URL.

Here is an example:

  • https://your-website.com/products#chat-open

B. A button class

To open the Slaask widget from a button, you just have to add the "slaask-open-widget" class to the button.

Here is an example:

  • <button type="button" class="slaask-open-widget">Click Me!</button>
