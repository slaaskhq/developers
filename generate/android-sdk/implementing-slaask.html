Implementing Slaask

  1. Locate your unique Slaask key.
  2. Add this library to your Android project.
  3. Verify that everything is working as it should.

Locate your unique Slaask key.

  • Sign in to your Slaask account at slaask.com.

  • Click on Personalization over on the left sidebar.

  • Copy your unique Slaask key (known as the Widget key) that corresponds to the Slaask live chat widget you wish to deploy on your mobile app.

Add this library to your Android project.

  • Download slaask.aar by clicking here

  • Open File -> New -> New Module -> Import JAR/ARR Package -> select slaask.aar

  • Open File -> Project Structure -> YourApplication Module -> Dependencies -> Add -> Module Dependency -> select slaask

  • Permissions 

    
<uses-permission android:name="android.permission.CAMERA"></uses-permission>
<uses-permission android:name="android.permission.INTERNET"></uses-permission>
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"></uses-permission>

    Starting with Android 6 Marshmallow (API 23) allow runtime permissions for CAMERA and WRITE_EXTERNAL_STORAGE

  • Add the following code to the MainActivity class:

    
import com.slaask.lib.model.SLAASKConfig;
import com.slaask.lib.model.SLAASKIdentify;
import java.util.HashMap;

public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        String token = "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx";

        SLAASKConfig config = new SLAASKConfig(token);

        SLAASKManager.getInstance()
                .setContainerId(R.id.container)
                .setContainerActivity(this)
                .setConfig(config);

        setContentView(R.layout.activity_main);
    }

    @Override
    protected void onActivityResult(int requestCode, int resultCode, Intent data) {
        super.onActivityResult(requestCode, resultCode, data);
        SLAASKManager.getInstance().onActivityResult(requestCode, resultCode, data);
    }
}

Verify that everything is working as it should.

Once you've built your app code (including the Slaask snippet):

  • Go to a view of your app where the show() method is called. The Slaask widget should appear.

  • Type something (such as: "Hey! Do you read me?").

  • Open up your Slack (exactly the same way you would when using Slack for normal communication purposes). Your message should appear in a dedicated channel on your Slack.

  • If you can see your message appearing within a channel, this means that your Slaask integration was a success. Congrats! Otherwise, if you believe you've followed this process correctly and that Slaask isn't working on your mobile app as it should, please contact us so that we may help you resolve the issue.
