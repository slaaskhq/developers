Implementing Slaask

  1. Locate your unique Slaask key.
  2. Add this library to your iOS project.
  3. Verify that everything is working as it should.

Find your Slaask code snippet.

  • Sign in to your Slaask account at slaask.com.

  • Click on Personalization over on the left sidebar.

  • Copy your unique Slaask key (known as the Widget key) that corresponds to the Slaask live chat widget you wish to deploy on your mobile app.

Add this library to your iOS project.

  • Download slaask.framework by clicking here.

  • Open Target Project -> Embedded Binaries -> select slaask.framework. Fill in the check box marked copy items if needed.

  • For projects developed in Objective-C, open General -> Build Settings find Always Embed Swift Standard Libraries and choose value YES

  • For iOS 10 and later versions, it’s necessary to fill the following fields in Info.plist:

    - Privacy - Photo Library Usage Description
    - Privacy - Camera Usage Description

  • Add the following code to the project AppDelegate’s method didFinishLaunchingWithOptions:

    (If not already replaced, replace "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" with your Widget Key.)

    
let config      = SLAASKConfig.init(slaaskToken: "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx")
let manager     = SLAASKManager.sharedInstance
manager.config  = config
                
    
SLAASKConfig *config    = [[SLAASKConfig alloc] initWithSlaaskToken:@"xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"];
SLAASKManager *manager  = [SLAASKManager sharedInstance];
manager.config          = config;

Verify that everything is working as it should.

Once you've built your app code (including the Slaask snippet):

  • Go to a View of your app where the show() method is called. The Slaask widget should appear.

  • Type something (such as: "Hey! Do you read me?").

  • Open up your Slack (exactly the same way you would when using Slack for normal communication purposes). Your message should appear in a dedicated channel on your Slack.

  • If you can see your message appearing within a channel, this means that your Slaask integration was a success. Congrats! Otherwise, if you believe you've followed this process correctly and that Slaask isn't working on your mobile app as it should, please please contact us so that we may help you resolve the issue.
