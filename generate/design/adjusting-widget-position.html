Adjusting the position of the widget's button

If you would like to adjust the position of your widget's button, because it hides some content behind, or for any other reason, you can try adding this CSS code to your webpage: 

<style>
  #slaask-button, #slaask-message {
    transform: translate(XXXpx, YYYpx);
  }
</style>

Replace the XXX and YYY by how many pixels you want to move your button.
