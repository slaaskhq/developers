Adjusting your widget's icon position

If you would like to adjust your widget's icon position in case it's hiding content or for any other reason, you can add this CSS code to your webpage: 

<style>
  #slaask-button, #slaask-message {
    transform: translate(XXXpx, YYYpx);
  }
</style>

Replace the XXX and YYY with the number of pixels you'd like to move your widget's icon.
