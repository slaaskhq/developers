Removing the white background from the button

Whe putting a transparent image on the Slaask button, you may want to remove the white background.

Here is a piece of CSS code you can add in a <style> tag to do so:


#slaask-button {
  -webkit-box-shadow: none!important;
  box-shadow: none!important;
}

#slaask-button .slaask-button-image {
  border: 0;
  background-color: transparent;
}
