Manage my Knowledge Base through the REST API

  1. GET all my Knowledge Base Entries
  2. GET a single Knowledge Base Entry
  3. CREATE a Knowledge Base Entry
  4. UPDATE a Knowledge Base Entry
  5. DELETE a Knowledge Base Entry

First, learn how to find your secret key.

GET all my Knowledge Base Entries

Endpoint

GET https://slaask.com/api/faqs

Parameters

Key Type Optional?
secret_key String No

Request


curl -X GET "https://slaask.com/api/faqs?secret_key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
            

OkHttpClient client = new OkHttpClient();

Request request = new Request.Builder()
  .url("https://slaask.com/api/faqs?secret_key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx")
  .get()
  .build();

Response response = client.newCall(request).execute();
            

var data = null;

var xhr = new XMLHttpRequest();
xhr.withCredentials = true;

xhr.addEventListener("readystatechange", function () {
  if (this.readyState === 4) {
    console.log(JSON.parse(this.responseText));
  }
});

xhr.open("GET", "https://slaask.com/api/faqs?secret_key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx");

xhr.send(data);
            

var request = require("request");

var options = { method: 'GET',
  url: 'https://slaask.com/api/faqs',
  qs: { secret_key: 'xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx' };

request(options, function (error, response, body) {
  if (error) throw new Error(error);

  console.log(body);
});
            

setUrl('https://slaask.com/api/faqs');
$request->setMethod(HTTP_METH_GET);

$request->setQueryData(array(
  'secret_key' => 'xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx'
));

try {
  $response = $request->send();

  echo $response->getBody();
} catch (HttpException $ex) {
  echo $ex;
}
            

import requests

url = "https://slaask.com/api/faqs"

querystring = {"secret_key":"xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"}

response = requests.request("GET", url, headers={}, params=querystring)

print(response.text)
            

require 'uri'
require 'net/http'

url = URI("https://slaask.com/api/faqs?secret_key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx")

http = Net::HTTP.new(url.host, url.port)
http.use_ssl = true
http.verify_mode = OpenSSL::SSL::VERIFY_NONE

request = Net::HTTP::Get.new(url)

response = http.request(request)
puts response.read_body

Response

Status: 200 OK 


{
"success": true,
"results": [
  {
    "id": 1,
    "title": "My Knowledge base title",
    "content": "Lorem ipsum dolor sit amet, consectetur adipiscing elit.",
    "created_at": "2016-10-12T15:22:09.000Z",
    "updated_at": "2016-10-12T15:22:09.000Z"
  }
]
}

Errors

Status: 403 Forbidden 


{
  "success": false,
  "message": "Invalid Widget secret key"
}

CREATE a Knowledge Base Entry

Endpoint

POST https://slaask.com/api/faqs

Parameters

Key Type Optional?
secret_key String No
title String No
content Text No

Request

Title is the title of the Knowledge Base Entry you want to create.
Content is the content of the Knowledge Base Entry you want to create. 


curl -X POST "https://slaask.com/api/faqs?secret_key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx&title=title&content=content"
            

OkHttpClient client = new OkHttpClient();

Request request = new Request.Builder()
  .url("https://slaask.com/api/faqs?secret_key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx&title=title&content=content")
  .post(null)
  .build();

Response response = client.newCall(request).execute();

UPDATE a Knowledge Base Entry

Endpoint

PUT https://slaask.com/api/faqs/ID

Parameters

Key Type Optional?
secret_key String No
id Integer No
title String No
content Text No

Request

ID is the id of the Knowledge Base Entry you want to update.
Title will be the new title of the Knowledge Base Entry you want to update.
Content will be the new content of the Knowledge Base Entry you want to update.
