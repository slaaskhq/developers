Initializing the widget

  1. Find your Slaask code snippet.
  2. Paste your Slaask code snippet.
  3. Verify that everything is working as it should.

Find your Slaask code snippet.

Once you are connected to your Slaask account on slaask.com, click on Widgets on the left sidebar.

Find your Slaask code snippet in this "Widgets" section. It will be located inside of an input on the right-hand side of the screen.

This input contains a unique Slaask key that corresponds to one of your Slaask widgets.

Paste your Slaask code snippet.

  • Download slaask.aar by clicking here

  • Open File -> New -> New Module -> Import JAR/ARR Package -> select slaask.aar

  • Open File -> Project Structure -> YourApplication Module -> Dependencies -> Add -> Module Dependency -> select slaask

  • Permissions 

    
<uses-permission android:name="android.permission.CAMERA"></uses-permission>
<uses-permission android:name="android.permission.INTERNET"></uses-permission>
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"></uses-permission>

    Starting with Android 6 Marshmallow (API 23) allow runtime permissions for CAMERA and WRITE_EXTERNAL_STORAGE

  • Add the following code to the MainActivity class:

    
import com.slaask.lib.model.SLAASKConfig;
import com.slaask.lib.model.SLAASKIdentify;
import java.util.HashMap;

public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        String token = "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx";

        SLAASKConfig config = new SLAASKConfig(token);

        SLAASKManager.getInstance()
                .setContainerId(R.id.container)
                .setContainerActivity(this)
                .setConfig(config);

        setContentView(R.layout.activity_main);
    }

    @Override
    protected void onActivityResult(int requestCode, int resultCode, Intent data) {
        super.onActivityResult(requestCode, resultCode, data);
        SLAASKManager.getInstance().onActivityResult(requestCode, resultCode, data);
    }
}

Verify that everything is working as it should.

Once you've built your app code (including the Slaask snippet):

  • Go to a view of your app where the show() method is called. The Slaask widget should appear.

  • Type something (such as: "Hey! Do you read me?").

  • In a new tab, open your Slack app (exactly the same way you would when using Slack for normal communication purposes). Your message should appear in a dedicated channel on your Slack app.

  • If you can see your message in a channel, it means your Slaask integration is a success! Congratulations! Otherwise, if you believe you've followed this process correctly and it still isn't working, please contact us so that we can help you resolve the issue.
