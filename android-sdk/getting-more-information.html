Gathering more information about your visitors

These instructions will walk you through the installation of Slaask on your app for logged-in users of your product. With this feature, you can track account data and have conversations with your users via Slaask.


import com.slaask.lib.model.SLAASKConfig;
import com.slaask.lib.model.SLAASKIdentify;
import java.util.HashMap;

public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        String name           = "name";
        String mVisitor_token = "a-unique-visitor-attribute";
        String token          = "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx";

        HashMap mOptions = new HashMap<>();

        mOptions.put("user_id" , "1234567890");
        mOptions.put("email" ,"user@gmail.com");

        SLAASKIdentify identify = new SLAASKIdentify(name, mOptions);
        SLAASKConfig config     = new SLAASKConfig(token, identify);

        config.setVisitorToken(mVisitor_token);

        SLAASKManager.getInstance()
                .setContainerId(R.id.container)
                .setContainerActivity(this)
                .setConfig(config);

        setContentView(R.layout.activity_main);
    }

    @Override
    protected void onActivityResult(int requestCode, int resultCode, Intent data) {
        super.onActivityResult(requestCode, resultCode, data);
        SLAASKManager.getInstance().onActivityResult(requestCode, resultCode, data);
    }
}

Make sure you replace the user data example (highlighted in pink) with your own code that inserts real user data for the currently logged-in user on each of Activity classes.

Replace a-unique-visitor-attribute (mVisitor_token) with a UNIQUE identifier for your visitor. This attribute will be used to identify your visitors in order to retrieve their conversation history and information.
