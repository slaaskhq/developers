Opening your widget from an item/button

To code a button or a menu navigation item on your site that will open the Slaask widget, you have 2 choices:

From an URL

To automatically open the Slaask widget when someone lands on a page, please add "#chat-open" to the end of your URL.

Here is an example:

https://your-website.com/products#chat-open

From a CSS Class

To open the Slaask widget from a button, you just have to add the slaask-open-widget class to the button.

Here is an example:

<a href="#" class="slaask-open-widget">Click Me!</a>

Note: the slaask-open-widget ID also works in all the HTML tags accepting the "id" parameter.

From a CSS ID

To open the Slaask widget from a button, you just have to add the slaask-open-widget id to the button.

Here is an example:

<button type="button" id="slaask-open-widget">Click Me!</button>

Note: the slaask-open-widget ID also works in all the HTML tags accepting the "id" parameter.
