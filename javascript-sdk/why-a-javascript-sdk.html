Why do we use Javascript for our API?

Our Javascript API gives you the ability to interact with your Slaask widget.

Your Slaask widget is auto-synchronized with our servers. Therefore, your personal widget configuration will always be loaded.

Your Slaask widget is initialized under the window global object. Just call our widget Javascript API using window._slaask global object or simply _slaask to get access to the module that includes the javascript widget..

Use _slaask.identify() to send personalized information about your visitors to the Slaask servers. You will then be able to retrieve this personalized information by asking for it directly in a Slack channel.
