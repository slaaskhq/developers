Events

Use this code snippet to listen to audio recordings of Slaask events:

document.addEventListener('slaask.open', function (e) {
    console.log(e.type + ' - by: ' + e.detail.by);
}, false);

Available events:

Event type Description Event details
ready When the widget is loaded and connected to the backend null
open When the widget is open { 'by': 'new_message' } in ['user', 'external_link', 'new_message']
close When the widget is closed { 'by': 'new_message' } in ['user', 'external_link', 'new_message']
sent_message When the user has sent a message { 'message': 'Hello', 'from': guest_id }
