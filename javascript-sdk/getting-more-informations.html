Gathering more information about your visitors

If you are using Slaask while your visitors are logged in, we provide a useful way for you to gather more information about them (name, email, new/returning visitor, etc.). To activate this feature, use the code snippet on the right:

<script src='https://cdn.slaask.com/chat.js'></script>
<script>
    _slaask.identify('JackieChan', {
        user_id: 1234,
        email: 'jackie@chan.com',
        speciality: 'martial artist'
    });
    _slaask.init('xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx', , { visitor_token: 'An-unique-visitor-attribute' });
</script>

You should replace An-unique-visitor-attribute (the second attribute of the _slaask.init function) with a UNIQUE identifier for your visitor. The attribute you come up with will be used to identify your visitor in order to retreive their conversation history as well as user information.
