Limiting Slaask to a few pages

There are two options for you to limit Slaask's presence to just a few pages or only one page on your site, for example, your landing page.

Option 1

You can include the Slaask snippet on the pages where you would like to display the Slaask widget.

This will give you the ability to only show the Slaask widget on some pages of your site.

For example, some of the Slaask users are only showing the widget to their logged-in users. In that case, other visitors have no access to their live support unless they log in.

Option 2

You can integrate the following code everywhere on your website.

In this code snippet you can add the urls of the pages where you want your Slaask livechat to appear. In other words, your Slaask livechat will not appear on pages that are not included in this code snippet. E.g.: https://get.slaask.com, https://get.slaask.com/pricing

<script type="text/javascript">
  // You can remove or add links here to load Slaask only on specific pages.

  var pages = [
    "https://get.slaask.com/",
    "https://get.slaask.com/pricing/",
    "https://get.slaask.com/features/"
  ];

  for (var i = 0; i < pages.length; i++) {
    if(window.location.href.indexOf(pages[i]) > -1) {
      _slaask.init('xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx');
    }
  }
</script>
