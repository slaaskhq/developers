Gathering more information about your visitors

If you are using Slaask while your visitors are logged in, we have provided a way for you to gather more information about them (name, email, new or returning visitor, etc.). Use the code snippet on the right to enable this feature:

<script src='https://cdn.slaask.com/chat.js'></script>
<script>
    _slaask.identify('JackieChan', {
        user_id: 1234,
        email: 'jackie@chan.com',
        speciality: 'martial artist'
    });
    _slaask.init('xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx', { visitor_token: 'A-unique-visitor-attribute' });
</script>

Replace A-unique-visitor-attribute (the second attribute of the _slaask.init function) with a UNIQUE identifier for your visitor. This attribute will be used to identify your visitor in order to retrieve his conversation history and information.
