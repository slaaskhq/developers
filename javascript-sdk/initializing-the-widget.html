Initializing the widget

  1. Find your Slaask code snippet.
  2. Copy your Slaask code snippet.
  3. Paste your Slaask code snippet.
  4. Verify that everything is working as it should.

Find your Slaask code snippet.

Once you are connected to your Slaask account on slaask.com, click on Widgets on the left sidebar.

Find your Slaask code snippet in this "Widgets" section. It will be located inside of a box consisting of several lines of JavaScript. Everything inside of this box is your Slaask code snippet. It begins with <script> and ends with </script>.

Your code contains a unique Slaask key that corresponds to one of your Slaask widgets. You can not reuse the same code snippet on multiple domains.

Here is an example of a Slaask code snippet:

<script src='https://cdn.slaask.com/chat.js'></script>
<script>
    _slaask.init('xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx');
</script>

Copy your Slaask code snippet.

Copy the snippet. Do not edit any part of the snippet. Simply copy it. You may wish to first paste it into a text document if you are worried about accidentally losing or changing the snippet on your clipboard.

Paste your Slaask code snippet.

Paste your snippet (unaltered, in its entirety) onto every web page that would like to provide the Slaask customer chat tool to your visitors. We recommend you paste it directly before the closing </body> tag.

If you use templates to generate dynamic pages for your site (e.g. PHP, ASP, or a similar technology), you can paste the Slaask code snippet into its own file, then include it in your page header.

For developers: here is the Slaask GitHub account for more resources related to Slaask code snippets.

Verify that everything is working as it should.

Once you've uploaded your website code (including the Slaask snippet):

  • Go to your website as a visitor. The Slaask widget should appear in the bottom right-hand corner of your site.

  • Click on the widget and then type something (such as: "Hey! Do you read me?").

  • In a new tab, open your Slack app (exactly the same way you would when using Slack for normal communication purposes). Your message should appear in a dedicated channel on your Slack app.

  • If you can see your message in a channel, it means your Slaask integration is a success! Congratulations! Otherwise, if you believe you've followed this process correctly and it still isn't working, please contact us so that we may help you resolve the issue.
